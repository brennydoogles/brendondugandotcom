About this site

This is a rewrite of the legacy PHP/CodeIgniter site I used from 2015 - mid-2019. Since this site exists primarily for the purpose of housing my CV as well as other personal information that might help potential employers get a sense of who I am as a person, I wanted to ensure that the site is not only descriptive of my skills, but evidential of said skills. As such, I have rewritten the site using the technologies I use regularly in my career and have released the source code on Github.

This site uses the following technologies:

  • Java 12
  • Spring Boot
  • Thymeleaf
  • Twitter Bootstrap
  • Maven
  • Heroku
